# heap
### ✅ 우선순위큐란?
가장 우선순위가 높은 데이터를 뽑을 수 있는 자료구조

ex.
- stack : 가장 나중에 들어간 것
- queue : 가장 먼저 들어간 것
- heap : 가장 값이 큰 것


### ✅ 힙이란?
키 값의 크기가 우선순위가 되는 것


### ➰특징
- Full Binary Tree 로 구성됨
- 가장 큰 값은 Root / 부모 노드는 두 자식 노드보다 큰 값이어야 함


### Operation
- insert : 힙의 말단에 삽입 후, upHeap
- remove : Root를 제거한 후, 말단을 Root로 올리고, DownHeap


### 부모와 자식의 상관관계
- 부모노드 = 자식노드/2
- 왼쪽 자식 = 부모노드*2 / 오른쪽 자식 = 부모노드*2 +1
- 절반이 내부노드

* 내부노드 : 자식이 하나 이상인 것
* 외부노드 : 리프노드
